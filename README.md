# mrncstt.github.io

My personal blog. Yes, another developer blog. No, I won't apologize for it.

## What's here

Posts about data engineering, Spark, open-source contributions, and the occasional travel guide that nobody asked for but everyone secretly needs.

Written in a mix of English and Portuguese because being bilingual means you get to confuse readers in two languages.

## Tech stack

- **Astro** for static site generation (because Jekyll served well but it was time to move on)
- **Shiki** for syntax highlighting with dual dark/light themes
- **Buttondown** for the newsletter (subscribe or don't, no pressure)
- **GitHub Pages** + **GitHub Actions** for deploy

## Design decisions

The design is intentionally monochromatic and minimal. Four fonts doing the heavy lifting: Newsreader for headings, DM Sans for UI, Source Serif 4 for prose, and JetBrains Mono for code. No accent colors. Elegance through contrast and spacing.

Horizontal sticky TOC, sidenotes in the margin on desktop, CSS grid treemap for tags, and stats inspired by [Raymond Camden's stats page](https://www.raymondcamden.com/stats).

## Running locally

```bash
npm install
npm run dev
```
